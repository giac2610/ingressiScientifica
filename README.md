# App Ingressi Scientifica

Sviluppata in Ionic/Angular + Capacitor

## Build e run

1. Compila il sito Angular (crea i nuovi file JS/CSS):

     ```Bash
     npm run build
     ```

2. Copia i nuovi file dentro Android:

     ```Bash
     npx cap sync android
     ```

3. Lancia l'app sul tablet:

     ```Bash

     npx cap run android
     ```
